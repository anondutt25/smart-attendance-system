A full-stack web based SMART ATTENDANCE SYSTEM that records students's attendances, examination marks, calculates percentage of attendace & marks (individually). Built extensively with PHP, MYSQL, CSS & HTML

Features included ~

- Role-based login for Admin and Students
- Admin can add students, take attendance, and manage marks
- Auto-calculates attendance percentage and grades
- Generate attendance and marks reports instantly
- Students can view their own marks and profile securely
- Clean, modern web interface with sidebar navigation

Getting Started on the go ~

- Requirements: XAMPP installed on your computer
- Download or clone the project into C:/xampp/htdocs/smart-attendance/
- Open XAMPP and start Apache and MySQL simultaneously
- Go to http://localhost/phpmyadmin and create a database named attendance_db
- Import the attendance_db.sql file into that database
- Open your local browser and visit or type http://localhost/smart-attendance/
- Log in as Admin or Student to get started

Default database requires no password, just connect with the php/db and we're gppd to go.

Passwords are not end to end encrypted, causing a lapse in the system, it stores password as plain text and not secured.
